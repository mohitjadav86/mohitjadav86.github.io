About Me

I’m Mohit Jadav, a Flutter and IoT developer from Ahmedabad with 5+ years of experience turning ideas into polished, real-world apps. I enjoy crafting clean, cross-platform mobile apps, connecting hardware with software through IoT, and exploring creative tools like single-player games built with Flutter Flame.

My work spans everything from designing smooth UI/UX and custom animations, to backend APIs, cloud storage with Cloudflare R2/D1, and automating workflows using shell scripts. I care about writing modular, scalable code and delivering apps that feel reliable, modern, and user-friendly.

Beyond just coding, I’m driven by curiosity: always learning, experimenting, and pushing my skills to build better digital products — whether for clients, the Play Store, or my own projects.

Key Skills:

Flutter & Dart (cross-platform apps) IoT integration (Bluetooth, Wi-Fi, sensors) Backend development (PHP, SQL, REST APIs, Cloudflare R2/D1) UI/UX design & custom animations Game development with Flutter Flame Shell scripting & workflow automation (macOS) Git & GitHub Actions (version control & CI/CD)
